# 福井観光エリアマップ

このプロジェクトは、日本の福井県の観光スポットをインタラクティブな3D地形マップ上に可視化するWebアプリケーションです。

## デモ

[https://code4fukui.github.io/fukui-kanko-map/](https://code4fukui.github.io/fukui-kanko-map/)

## 機能

- [deck.gl](https://deck.gl/) を活用した福井県のインタラクティブな3D地形マップ。
- 観光スポットはアニメーション付きの3Dアイコンでマークされます。
- アイコンにホバーすると、ツールチップで場所の詳細を確認できます。
- アイコンをクリックすると、関連する統計ページが新しいタブで開きます。

## 要件

本プロジェクトを利用するには、WebGLをサポートするモダンなWebブラウザが必要です。

## 使用方法

Webブラウザで[ライブデモ](https://code4fukui.github.io/fukui-kanko-map/)を開いてください。

## データソース

データは、[福井県 観光/旅行サイト - ふくいドットコム](https://code4fukui.github.io/fukui-kanko-survey/area.csv)が提供するCSVファイルから取得しています。
