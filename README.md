# Fukui Tourism Area Map

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

This project is a web application that visualizes tourist spots in Fukui Prefecture, Japan on an interactive 3D terrain map.

## Demo

[https://code4fukui.github.io/fukui-kanko-map/](https://code4fukui.github.io/fukui-kanko-map/)

## Features

-   Interactive 3D terrain map of Fukui Prefecture powered by [deck.gl](https://deck.gl/).
-   Tourist locations are marked with animated 3D icons.
-   Hover over an icon to view location details in a tooltip.
-   Click an icon to open a related statistics page in a new tab.

## Requirements

This project requires a modern web browser with WebGL support.

## Usage

Open the [live demo](https://code4fukui.github.io/fukui-kanko-map/) in your web browser.

## Data Source

Data is sourced from a CSV file provided by the [Fukui Prefecture Tourism/Travel Site - Fuku-i.com (ふくいドットコム)](https://code4fukui.github.io/fukui-kanko-survey/area.csv).
